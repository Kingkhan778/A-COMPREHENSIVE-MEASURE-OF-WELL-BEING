About HDI Nexus

HDI Nexus is a Machine Learning based analytics platform that estimates the Human Development Index (HDI) using key socioeconomic indicators.


📌 Project Overview

The application predicts Human Development Index values using a Linear Regression Machine Learning model trained on official HDI datasets. The prediction considers Life Expectancy, Education Indicators and Gross National Income.
